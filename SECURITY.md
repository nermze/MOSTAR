# Security Policy

### Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.0   | :white_check_mark: |
| < 1.0.0   | :x:                |

### Reporting a Vulnerability

If you discover a security vulnerability (such as unsafe dependency execution, privilege escalation, or data handling flaws), please let us know right away.
Please do not report security vulnerabilities through public GitHub issues.

Instead, please follow this process:
* Send an email to nermze@gmail.com with the subject line "Mostar Security Vulnerability".
* Provide details: Include a clear description of the vulnerability, the version of Mostar you are using, your environment (e.g., Conda, macOS/Linux), and steps to reproduce the issue.
